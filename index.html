The cloud version of SeatShare is no longer available as of January 2019.

When we started in 2014, the goal was to solve a particular problem we had: make sure our extra seats at Nashville Predators games never went empty. More than 150 people from all over the country used (or at least tried) SeatShare, and we're pretty proud of that.

SeatShare lives on as an open source project, which means anyone can take the code we wrote and run their own private season ticket group. We hope you found it useful and thank you for giving us a try.

All user data hosted on this website will be deleted March 31, 2019. If you would like a copy of your data before it is deleted, you can reach out to us by email.
